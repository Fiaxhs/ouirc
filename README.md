###Setup
- `git clone https://github.com/Fiaxhs/ouirc`
- `cd ouirc`
- `npm install`
- open `lib/app.js` and edit `Ouirc.prototype.connectServers` according to comments.
- `make` should launch app. (the `nodewebkit` binary should be in your path)
